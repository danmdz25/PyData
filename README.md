# 📊 PyData  
A collection of Python studies focused on data manipulation and analysis using:  

- **NumPy** – Efficient numerical computing and array operations  
- **Pandas** – Data manipulation and analysis tools  
- **Matplotlib** – Data visualization and plotting  

This repository contains scripts and Jupyter notebooks with practical examples and exercises to reinforce key concepts.  

## 🚀 Topics Covered  
✅ NumPy: Arrays, indexing and mathematical operations  
✅ Pandas: DataFrames, series, data cleaning, and transformations  
✅ Matplotlib: Plotting, customizing visualizations, and styling  

## 📂 Repository Structure  
- study/ – Main directory
- Matplotlib/ – Matplotlib exercises 
- Numpy/ – Numpy exercises 
- Pandas/ – Pandas exercises

## 🛠️ Requirements  
To run the code, install the required libraries:  

bash
pip install numpy pandas matplotlib
